# Computer-Vision_QR-reader

License plate detection with Python and yolov3 !

## 🔨 Environment
- python3.11.7
- You Only Look Once (YOLO) v.3
- requirements.txt

--------------------------
You can download the weights file from this [link](https://drive.usercontent.google.com/download?id=1vXjIoRWY0aIpYfhj3TnPUGdmJoHnWaOc&export=download&authuser=0).

## Test on a single image:

```python
python object_detection_yolo.py --image-dir=./image-directory
```

## Test on a single video file:
```python
python object_detection_yolo.py --video=cars.mp4
```

## Test on the webcam:
```python
python object_detection_yolo.py
```
